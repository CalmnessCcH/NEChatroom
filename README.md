# NEChatroom

基于网易云信新一代（G2）实时音视频SDK的多人语音聊天室Demo，包含了实时音频通话、麦序管理、伴音、聊天室等功能。本项目提供源码及文档说明。

## 支持的平台

多人语音聊天室Demo包含 Android, iOS平台。

多人语音聊天室Demo，展示了：

* 加入/离开房间；
* 实时音频通话；
* 聊天室；
* 麦序管理、伴音等；
